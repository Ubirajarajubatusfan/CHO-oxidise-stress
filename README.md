# Поиск генов-мишеней для повышения стрессоустойчивости клеточной линии-продуцента

### Партнерская организация (модельный индустриальный заказчик)

ООО «CellPharm Solutions» — разработка клеточных линий для производства рекомбинантных белков

### Проблема

Компания тестирует клеточную линию млекопитающих CHO (chinese hamster ovary cell - клетки яичника китайского хомячка), используемую для получения рекомбинантного белка. На этапе масштабирования наблюдается снижение жизнеспособности клеток под действием окислительного стресса, сильно снижается выход белка 

### Цель

На основе RNA-seq-данных выявить гены и пути, связанные со стрессовым ответом, и предложить обоснованный список генов-мишеней кандидатов для дальнейшей генной модификации.

### Задачи (более подробные скрипты и пайплайн в соответствующих файлах)

•	Провести контроль качества ридов и кратко интерпретировать результаты QC.

•	Выполнить картирование ридов на референсный геном и получить матрицу счетов.

•	Провести анализ дифференциальной экспрессии в R (DESeq2 или edgeR).

•	Выполнить GO/KEGG enrichment и показать биологическую интерпретацию результатов.

•	Сформировать shortlist из 3–5 генов-мишеней для инженерной модификации (knockout, knockdown, overexpression) с аргументацией.

•	Описать ограничения анализа и риски переноса результатов в мокрую лабораторию.

### Исходные данные

•	RNA-seq-данные контрольной и стрессовой групп (3 биологических реплики для контроля (день 0 без окислительного стресса) и 3 реплики для основной группы (день 5 окислительного стресса)).

•	Референсный геном и аннотация генов.

•	Метаданные эксперимента и краткое описание биологической модели.

### Результаты

• матрица counts

• Графики GO/KEGG enrichment

• Shortlist из 5 генов-мишеней, влияние на которых в теории может ослабить последствия окислительного стресса
