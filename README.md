# Поиск генов-мишеней для повышения стрессоустойчивости клеточной линии-продуцента

### Партнерская организация (модельный индустриальный заказчик)

ООО «CellPharm Solutions» — разработка клеточных линий для производства рекомбинантных белков

### Проблема

Компания тестирует клеточную линию млекопитающих CHO (chinese hamster ovary cell - клетки яичника китайского хомячка), используемую для получения рекомбинантного белка. На этапе масштабирования наблюдается снижение жизнеспособности клеток под действием окислительного стресса, сильно снижается выход белка
